# Macro
Macro made in python that uses ctypes, only recognises a-z, 1-0, space, enter and backspace keys

# Usage

## Record events

'''bash
python macro.py --record
'''

## Run events

### Repeating inputs

'''bash
python macro.py -f out.py -t 
'''

### run once

'''bash
python macro.py -f out.py
'''
